# even-seven-super-logic
C program using loop, if-else, continue, and break to print special outputs for numbers divisible by 2 and 7 while skipping multiples of 5.


# Even Seven Super Logic

## Description
This C program prints numbers from 1 to 100 using loops and conditional statements.

##  Features
- Stops the loop at 90
- Skips numbers divisible by 5
- Prints "Super" for numbers divisible by both 2 and 7
- Prints "Even" for numbers divisible only by 2
- Prints "Seven" for numbers divisible only by 7
- Prints the number otherwise

##  Concepts Used
- for loop
- if else if
- continue
- break
- logical AND operator (&&)

##  Example Output
14 → Super  
8 → Even  
7 → Seven
